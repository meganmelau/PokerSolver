The mark43 3-Poker uses Python programming language, Version 3.7.4.

To test with your own test case in text file
Example:
python mark43-main.py  tests/tie_breaker_test

To run with test script
./run_tests3 "python mark43-main.py"

Assumptions:
This assumes the inputs are well-formed
- unique player id
- cards are 2-9,A,J,Q,K
- suits are c, d, h, s
- hand is player id, card1, card2, card3
